# Expression Calculator

Калькулятор математических выражений с поддержкой переменных, функций и операций.
Подробнее о реализации в JavaDoc https://letnapan.github.io/laba_expression/
## Возможности
### Числа
- Целые: `42`, `5`
- Дробные: `3.14`, `0.5`

### Арифметические операции
- Сложение: `+`
- Вычитание: `-`
- Умножение: `*`
- Деление: `/`

### Тригонометрические функции
- `sin(x)` — синус x (радианы)
- `cos(x)` — косинус x (радианы)
- `tan(x)` — тангенс x (радианы)

### Логарифмические функции
- `ln(x)` — натуральный логарифм
- `log10(x)` — логарифм по основанию 10
- `log2(x)` — логарифм по основанию 2

### Другие функции
- `sqrt(x)` — квадратный корень
- `abs(x)` — модуль числа

### Переменные
- Одна буква: `x`, `y`, `a`
- Значения передаются через `VariableProvider`

### Обработка ошибок
- Синтаксические ошибки (некорректная структура выражения)
- Деление на ноль
- Неизвестные функции
- Неизвестные переменные
- Некорректные аргументы функций (например, логарифм от отрицательного числа)

## Установка

### Требования
- Java 11 или выше 
- Maven 3.6 или выше

### -----

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/letnapAn/laba_expression.git
   cd labaExpression
   ```
2. **Проверьте версию Java**
   ```bash
   java -version
   ```
3. **Скомпилируйте проект**
   ```bash
   mvn clean compile
   ```
4. **Соберите JAR-файл**
   ```bash
   mvn package   
   ```
